# EkstreHub

EkstreHub, kredi karti ekstrelerini tek yerde toplayip raporlamayi hedefleyen bir projedir.

Bu repo ilk iskelet olarak hem Home Assistant add-on hem standalone calisma modelini destekleyecek sekilde hazirlanmistir.

## Hedefler

- PDF/CSV ekstre yukleme
- E-posta kutusundan otomatik ekstre toplama
- Kategori bazli harcama analizi
- Aylik ve kart bazli raporlama

## Gelistirme Planı (ilk asama)

1. API health endpoint
2. Basit UI shell
3. Ekstre ingest pipeline taslagi
4. HA add-on metadata ve standalone docker calisma akisi
